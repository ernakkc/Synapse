Synapse

AI-powered assistant with intelligent memory system

View on GitHub Get Started

✨ Features

🧠 Intelligent Memory

Three-tier memory system with SQLite persistence: short-term, long-term, and episodic memory for context-aware interactions.

🤖 AI-Powered Execution

Natural language to system commands with support for Ollama, OpenAI, and Anthropic models.

💬 Smart Conversations

Context-aware chat with intent detection and advanced NLP capabilities.

📊 Action Planning

Intelligent task breakdown and execution with multi-step workflows.

🖥️ Cross-Platform

Full support for Windows, macOS, and Linux systems.

🏗️ Clean Architecture

Built with DDD principles and separation of concerns for maintainability.

🚀 Getting Started

Prerequisites

  • Node.js ≥18.0.0
  • npm ≥8.0.0
  • Ollama (if using local LLM)

Installation

# Clone the repository
git clone https://github.com/ernakkc/Synapse.git
cd Synapse

# Install dependencies
npm install

# Configure environment
cp .env.example .env

# Edit .env with your settings
nano .env

Quick Start

# Start Ollama (if using local LLM)
ollama serve

# Run CLI interface
npm run cli

# Or start the main application
npm start

🧠 Memory System

Three-Tier Architecture

  • Short-Term Memory: Temporary storage with TTL for current session context
  • Long-Term Memory: Persistent knowledge with verification and consolidation
  • Episodic Memory: Event sequences with emotional tracking and summarization

Database Schema

SQLite-based persistence with:

  • Foreign key constraints for data integrity
  • Indexes for fast query performance
  • Views for common queries
  • Automatic cleanup of expired memories

🤖 AI Providers

Synapse supports multiple LLM providers:

Ollama (Local)

LLM_PROVIDER=ollama
OLLAMA_MODEL=llama2
OLLAMA_BASE_URL=http://localhost:11434

OpenAI

LLM_PROVIDER=openai
OPENAI_API_KEY=your-api-key
OPENAI_MODEL=gpt-4

Anthropic

LLM_PROVIDER=anthropic
ANTHROPIC_API_KEY=your-api-key
ANTHROPIC_MODEL=claude-3-opus-20240229
